#  程序结构

Python以缩进区分程序块

三种结构

* 顺序
* 循环
* 分支

## 顺序结构

依次按行执行

## 分支结构

### 基本结构

~~~python
if 条件表达式：
	语句1
    ……
语句1
~~~

注：字符串中只有为空，才为假

### 双向分支

~~~python
if 条件表达式：
	语句
    ……
else：
	语句
    ……
~~~

### 多路分支

~~~python
if 条件表达式：
	语句
elif 条件表达式：
	语句
    ……
else：
	语句
~~~

## 循环语句

重复执行某个任务

分类

* for循环
* while循环

### for循环

~~~python
for 变量 in 序列：
	语句
    ……
~~~

### for-else语句

for循环又是需要执行收尾工作，此时使用else

~~~python
for 变量 in 序列：
	语句
else：
	语句
~~~

### break，continue，pass

* break：无条件结束循环，简称循环猝死
* continue：继续
* pass：占位符

### while循环

~~~python
while 条件表达式：
	语句
~~~

### while-else语句

~~~python
while 条件表达式：
	语句
else：
	语句
~~~
